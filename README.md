# 🏗️ AJC Soluções Estruturais

> **Status do Projeto:** Concluído 🚀

Site institucional desenvolvido para a **AJC Soluções Estruturais**, empresa especializada em serralheria, solda industrial e manutenção predial em Biguaçu e toda a Grande Florianópolis.

---

## 🛠️ Tecnologias Utilizadas

Para este projeto, foquei em performance, responsividade e SEO:

- **HTML5 & CSS3:** Semântica avançada e estilização moderna.
- **Bootstrap 5:** Sistema de grid e componentes responsivos.
- **JavaScript (Vanilla):** Implementação de máscaras de formulário e integrações.
- **Intersection Observer API:** Animações de "reveal" ao rolar a página para melhor UX.
- **GitHub Actions:** Pipeline de **CI/CD** para deploy automatizado no GitHub Pages.
- **Sitemap & Robots.txt:** Otimização para indexação em motores de busca (SEO).

---

## ✨ Funcionalidades Principais

- [x] **Design Adaptativo:** Experiência fluida em Desktop, Tablet e Mobile.
- [x] **Acessibilidade:** Uso de ARIA labels e contraste de cores adequado.
- [x] **Formulário Inteligente:** Envio de orçamentos formatados direto para o WhatsApp do cliente.
- [x] **Otimização de Performance:** Imagens em formato `.webp` e carregamento assíncrono.
- [x] **Open Graph:** Preview profissional para compartilhamento em redes sociais.
